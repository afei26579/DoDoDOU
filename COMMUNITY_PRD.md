# 温馨手工坊 (Cozy Studio) - 完整产品设计文档 (PRD)

## 1. 产品概述
**产品定位**：一款专为拼豆（Perler Beads）爱好者设计的治愈系辅助工具，提供从灵感生成到物理制作的全流程数字化支持，优先手机移动端项目。
**核心价值**：通过图片转图纸、AI灵感生成、精准参数控制和沉浸式跟做模式，降低手工门槛，提升创作专注度。
**视觉风格**：温馨手工坊 (Cozy Studio)。主打低饱和度马卡龙色系、圆润卡片设计、漫反射软阴影，营造温暖、亲切、治愈的手作氛围。

---

## 2. 视觉规范 (Design System)
### 2.1 色彩方案
- **主色 (Primary)**：`#D8B4E2` (柔和紫) - 用于核心按钮、激活导航、重要标识。
- **背景 (Background)**：`#FDFBF7` (暖米色) - 全局底色，模拟手工纸张的温馨感。
- **卡片 (Surface)**：`#FFFFFF` (纯白) - 内容容器，配合柔和阴影。
- **文字 (Text)**：`#5D534A` (摩卡棕) - 标题与正文色，比纯黑更温润。
- **点缀色**：`#B5EAD7` (薄荷绿) - 用于图片上传卡片、成功状态；`#FFDAC1` (暖橙) - 用于高亮提醒。

### 2.2 字体规范
- **标题**：`ZCOOL KuaiLe` (站酷快乐体) - 营造活泼、具有亲和力的品牌个性。
- **正文**：`Varela Round` / `Nunito` - 具有圆角终端的无衬线体，阅读体验极佳。

### 2.3 UI 元素
- **圆角**：容器 24px，小元素 16px。
- **阴影**：大扩散、低浓度的彩色投影，增加浮动感。
- **导航图标**：手绘质感图标，选中状态为紫色填充。

## 3. 页面详细设计 (9 核心页面)

### 3.1 发现 (Home) - {{DATA:SCREEN:SCREEN_34}}
- **核心功能**：创作入口、进度管理、灵感启发。
- **设计细节**：
  - **顶部**：治愈系问候语与萌系头像。
  - **操作区**：并排的“照片变图纸”（薄荷绿）与“AI灵感生成”（紫色）大卡片。
  - **进度区**：横向滚动的“继续拼豆”项目卡片，显示完成百分比。
  - **灵感画廊**：底部新增自动左右滚动的热度图纸展示区。

### 3.2 图片裁剪 (Crop) - {{DATA:SCREEN:SCREEN_43}}
- **核心功能**：素材预处理。
- **设计细节**：
  - **布局**：图片区域居中显示，最大化视觉空间。
  - **工具栏**：精简为“旋转”、“水平镜像”、“垂直镜像”、“重置”四个核心功能。
  - **交互**：点击“下一步”直接跳转至工坊参数设置页。

### 3.3 工坊 - 参数设置 (Settings) - {{DATA:SCREEN:SCREEN_77}}
- **核心功能**：图纸生成参数配置。
- **设计细节**：
  - **图片预览**：顶部显示裁剪后的原图。
  - **参数组**：纵向排版，包含画布尺寸（15x15至48x48）、品牌色板（Perler, Hama等）、风格选择（写实、卡通、精简）及颜色合并阈值滑块。
  - **移动端优化**：间距适中，确保在手机屏幕无需过多滑动即可看全参数。

### 3.4 工坊 - 图纸预览 (Preview) - {{DATA:SCREEN:SCREEN_31}}
- **核心功能**：转换效果确认。
- **设计细节**：
  - **像素预览**：图片占位替换为生成的像素风格图纸。
  - **折叠面板**：下方的参数设置收纳在带小三角的折叠栏中。
  - **统计数据**：展示使用的颜色数及总豆豆数量。
  - **操作矩阵**：提供一键去背景、下载图纸、手动编辑、立即拼豆四个入口。

### 3.5 图纸编辑 (Editor) - {{DATA:SCREEN:SCREEN_57}}
- **核心功能**：图纸细节微调。
- **设计细节**：
  - **画布**：全屏像素网格，支持缩放。
  - **预览浮窗**：右上角可拖动的小型全局预览窗。
  - **工具组**：左侧工具栏包含铅笔、橡皮、油漆桶、吸色器、撤销/重做。
  - **操作区**：底部设有“开始拼豆”和“一键去背景”按钮，顶部为保存与下载。

### 3.6 沉浸拼豆 (Focus Mode) - {{DATA:SCREEN:SCREEN_1}}
- **核心功能**：辅助物理拼豆制作。
- **设计细节**：
  - **沉浸态**：使用深色背景（#463E38），图纸居中且左右宽度最大。
  - **层级引导**：仅当前层颜色高亮，其余 20% 透明。
  - **策略选择**：底部可选边缘优先、大块优先、距离优先等拼豆策略。
  - **物料看板**：实时显示当前层所需色号及具体数量。

### 3.7 下载设置 (Download) - {{DATA:SCREEN:SCREEN_8}}
- **核心功能**：导出参数配置。
- **设计细节**：
  - **形态**：紧凑型弹窗设计。
  - **配置项**：作者署名、分享码、网格线开关/间隔/颜色、用料清单、水印添加。
  - **精简**：移除了坐标数字和水平镜像，提升导出效率。

### 3.8 作品集 (Collection) - {{DATA:SCREEN:SCREEN_39}}
- **核心功能**：个人图纸管理。
- **设计细节**：
  - **瀑布流**：展示保存和完成的作品，带有“已完成”、“草稿”状态标签。
  - **导航**：底部导航栏图标与风格高度统一，选中态为紫色。

### 3.9 灵感工坊入口 (Creator Hub) - {{DATA:SCREEN:SCREEN_23}}
- **核心功能**：独立的工坊入口。
- **设计细节**：
  - **留白美学**：中心采用轻量级插画引导，强调“开始你的创意旅程”。
  - **品牌色板**：直观展示当前选用的品牌配色，方便快速切换。

## 4. 核心交互流程
1. **生成流**：首页上传图片 -> 裁剪 -> 工坊设置参数 -> 生成预览。
2. **编辑流**：预览页进入编辑器 -> 细节微调 -> 保存或下载。
3. **制作流**：预览页或作品集 -> 点击“立即拼豆” -> 进入沉浸模式，按策略依次摆放豆子。
